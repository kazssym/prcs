 What is PFE?

PFE is a friendly front end to the PRCS version control system. It will consist mainly of a file browser and version tree tools. From within these tools, most of the PRCS operations can be performed, such as merging, displaying diffs, adding new files to projects, etc. Users should never have to enter the prcs info command or hand edit a .prj file again.
 

Okay, So What is PRCS?

PRCS is a pretty decent version control system that I find much better than CVS, RCS, or any of the others out there. You can look at the PRCS homepage. The biggest drawback I see with PRCS is a lack of a decent interface. PFE hopes to remedy that.
